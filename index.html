HTML Tutorial

HTML stands for Hyper Text Markup Languange. It is used to design web pages using markup language. HTML is the combination of Hypertext and Markup language. Hypertext defines the link between web pages.

CSS Tutorial

CSS stands for Cascading Style Sheets. It describes how HTML elements are to be displayed on screen, paper, or in other media. CSS also saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.
